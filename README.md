

*Extractor* is a static [dataflow analyzer](https://en.wikipedia.org/wiki/Data-flow_analysis) and program slicer for the [*R*](https://www.r-project.org/) programming language (currently tested for versions `4.x`).

## ⭐ Getting Started

To get started with _flowR_, please check out the Overview. 

```shell
docker run -it --rm extractor
 ```
